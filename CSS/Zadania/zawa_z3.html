Cechy modyfikowania formatowania w HTML oraz CSS

Tabela porównawcza

HTML CSS
Formatowanie bezpośrednio w znacznikach. Formatowanie oddzielone od struktury strony.
Mniej możliwości stylizacji. Bardzo duże możliwości zmiany wyglądu.
Kod strony jest dłuższy. Kod jest bardziej uporządkowany.
Trudniej zmieniać wygląd wielu elementów. Można zmienić wygląd wielu elementów jednym stylem.

Nowe możliwości CSS w porównaniu z HTML

  1. Oddzielenie wyglądu strony od jej struktury.
  2. Możliwość zmiany stylu wielu elementów jednocześnie.
  3. Większa kontrola nad czcionkami, kolorami i układem.
  4. Możliwość tworzenia animacji i efektów wizualnych.
  5. Lepsze dopasowanie strony do różnych urządzeń.

Wady CSS

  1. Różne przeglądarki mogą inaczej interpretować style.
  2. Wymaga dodatkowej nauki składni.
  3. Błędy w CSS mogą wpłynąć na wygląd całej strony.
