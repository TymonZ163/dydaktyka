Budowa selektora

    1. Dwie części selektora:
    Selektor CSS składa się z dwóch głównych części:
       • Selektor – wskazuje element HTML, który ma zostać ostylowany (np. p, h1, div).
       • Deklaracja – określa styl, jaki zostanie zastosowany do wybranego elementu.

    2. Znaczenie nawiasów klamrowych { }
    Nawiasy klamrowe zawierają deklaracje stylów. W środku zapisujemy cechy i wartości, które określają wygląd elementu.

    3. Cechy
    Cechy (właściwości) określają co chcemy zmienić w wyglądzie elementu, np. kolor tekstu, rozmiar czcionki lub tło. Przykłady: color, font-size, background.

    4. Wartości
    Wartości określają jak ma wyglądać dana cecha. Są przypisane do cechy po dwukropku. Np. w zapisie color: red; – red jest wartością.

    5. Znak średnika (;)
    Średnik stosujemy po każdej deklaracji (cecha: wartość). Przy ostatniej deklaracji w bloku nie jest konieczny, ale często i tak się go stosuje dla czytelności kodu.
