Sposoby definiowania (określania) kolorów

Sposób zapisu Przykład Opis
Nazwa koloru red Kolor zapisany za pomocą nazwy w języku angielskim.
HEX #FF0000 Zapis szesnastkowy koloru.
RGB rgb(255, 0, 0) Zapis koloru przy użyciu wartości czerwonej, zielonej i niebieskiej.
RGBA rgba(255, 0, 0, 0.5) RGB z dodatkową przezroczystością (alpha).
HSL hsl(0, 100%, 50%) Kolor określony przez odcień, nasycenie i jasność.
HSLA hsla(0, 100%, 50%, 0.5) HSL z dodatkową przezroczystością.
